

Linux中变量$#,$@,$0,$1,$2,$*,$$,$?的含义 - jihite - 博客园 
http://www.cnblogs.com/kaituorensheng/p/4002697.html


```sh
$# 是传给脚本的参数个数
$0 是脚本本身的名字
$1 是传递给该shell脚本的第一个参数
$2 是传递给该shell脚本的第二个参数
$@ 是传给脚本的所有参数的列表
$* 是以一个单字符串显示所有向脚本传递的参数，与位置变量不同，参数可超过9个
$$ 是脚本运行的当前进程ID号
$? 是显示最后命令的退出状态，0表示没有错误，其他表示有错误
```
