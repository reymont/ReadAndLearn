

二、子网掩码位数与子网掩码的计算

子网掩码的最大位数为32位，C类单个网段所容纳的最大IP数目为256，包括网络地址和广播地址。
　　
例：

192.168.0.1/27
32-27=5 （最大子网位数减去当前子网位数）
2的5次方为32
256-32=224
255.255.255.224为192.168.0.1/27的子网掩码

所以得出计算公式：

`子网掩码的尾数（255.255.255.X）=256－2的（32－掩码当前位数）次方`
